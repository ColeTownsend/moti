# moti 🐼

The universal React Native animation library.

```jsx
<View from={{ opacity: 0 }} animate={{ opacity: 1 }} />
```

Coming soon.

# Highlights

- Mount/unmount animations, like `framer-motion`
- Works on all platforms
- 60 FPS
- Powered by Reanimated 2
- Web support, out-of-the-box
- Expo support
