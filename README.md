# Highlights

- Universal: works on all platforms
- Mount/unmount animations, like `framer-motion`
- Performant animations at 60 FPS
- Powered by Reanimated 2
- Web support, out-of-the-box
- Expo support
- Intuitive API
- Variant support with `useAnimationState`
